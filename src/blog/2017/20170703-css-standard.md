# CSS 标准
为了维护且便于快速定位到指定样式，最好将样式表分为几块。规则是，把一般的规则放前面，越往后的规则越特殊。
#### 一般性样式
- 主体样式  
body标记的样式，以及一些站点内所有元素继承的样式。  
- 全局rest样式
- 链接
- 标题
- 其他元素
#### 辅助样式 
在整个站点内使用的一般类。  
- 表单
- 通知和错误
- 一致的条目

#### 页面结构
- 标题、页脚和导航
- 布局
- 其他页面结构元素

#### 覆盖
